## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/phanendharreddy/phanendharreddy.github.io.git

# Navigate to project directory
cd phanendharreddy.github.io

# Install dependencies
npm install
```

### Development

```bash
# Start the development server
npm start

# Build for production
npm run build
```

### Deployment

```bash
# Deploy to GitHub Pages
npm run deploy
```
